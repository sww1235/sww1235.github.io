File Format

The file format is using markdown, and is as follows.

  • The first line is the page title using a level 1 header with markdown syntax #

  • At the end, put a tag block, which is a code block using ```, with the language indication as tags

  • Headers shall use ATX markdown header synax (enforced by lint rules), and pandoc style header id tags {#tag}
